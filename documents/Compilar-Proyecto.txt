¿Cómo compilar el Proyecto?

El presente proyecto se compila y ejecuta ingresando en una consola que ya se encuentre dentro de la carpeta raiz del proyecto lo siguiente:

./compilar.sh

REQUISITOS:

Para que funcione correctamente el comando " ./compilar.sh " se debe de instalar make en windows, para tal fin por favor siga la siguiente guía : https://parzibyte.me/blog/2020/12/30/instalar-make-windows/

COMANDOS DISPONIBLES:

- ./compilar.sh     -> comando encargado de compilar y ejecutar el proyecto
- make              -> comando encargado solo de compilar el proyecto
- make clean        -> comando encargado de eliminar el ejecutable y los archivos pre-compilados del proyecto
